<!-- Smart Meal Planner (React Frontend)

Smart Meal Planner là một ứng dụng web frontend xây dựng bằng React, giúp người dùng:
- Lập kế hoạch ăn uống theo ngày / tuần
- Xem công thức nấu ăn và nguyên liệu cần thiết
- Quản lý nguyên liệu có sẵn tại nhà
- Tự động tạo danh sách đi chợ

Dự án được thực hiện nhằm luyện tập React core và thiết kế frontend theo tư duy sản phẩm, có định hướng mở rộng backend trong tương lai.

Công nghệ sử dụng
+ React (Vite)
+ JavaScript (ES6+)
+ React Router
+ Context API
+ localStorage (mock data)

Chức năng chính
+ Đăng nhập / đăng ký (frontend)
+ Xem danh sách & chi tiết công thức
+ Lập lịch ăn uống theo tuần
+ Gợi ý món từ nguyên liệu có sẵn
+ Tạo checklist đi chợ

Mục tiêu học tập
Component-based architecture
Quản lý state và logic nghiệp vụ
Tách UI – logic – data
Chuẩn bị nền tảng cho React + Spring Boot

Hướng phát triển
+ Backend: Spring Boot
+ Database: SQL Server
+ JWT Authentication

Tác giả: An Nguyễn Thanh
Mục đích: Portfolio & học tập React -->
